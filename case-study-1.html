Plastic on Earth Website

Category

Website Design

The Brief

Each year, plastic bags and other plastic waste are dumped into the ocean destroying as many as one million sea animals. Created a website to raise awareness about plastics' harmful effects. The idea was to create a count-down website that would hopefully reach a larger number of people. With the plastic-on-earth website, the goal was to create a more engaging and interactive experience, that educates users. The website will used Illustrator-created graphics and Javascript to create the website's base code and CSS for content placement.
